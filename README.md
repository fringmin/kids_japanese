# こどもにほんご — 兒童日文學習小遊戲

給小朋友（國小低中年級）的日文學習 PWA：50 音、主題單字卡、聽力遊戲、跟讀練習。
主題單字圍繞他們喜歡的世界觀（角落生物、幽靈繪本、狀聲詞、日本旅遊）。

## 功能

| 模式 | 內容 |
|---|---|
| 🗾 50音表 | 平假名／片假名切換，點假名發音＋例字 |
| 🃏 單字卡 | 四個主題，點卡片聽發音 |
| 👂 聽音選字 | 聽假名發音選正確的字（計分遊戲） |
| 🤝 平片配對 | 看平假名選對應的片假名（計分遊戲） |
| 🎧 聽單字選圖 | 聽單字選正確的圖（計分遊戲） |
| 🎤 跟著唸 | App 唸一次 → 小孩跟著唸，語音辨識給回饋（鼓勵導向） |

- 兩位玩家各自的星星、分數、學習記錄（存在裝置 localStorage）
- 答對 +10 分，連續答對 3 題以上每題再 +5
- 發音用裝置內建日文語音（iOS: Kyoko），離線可用
- 跟讀辨識用 Web Speech API（需網路與麥克風權限），不支援時自動退成錄音回放

## 裝到 iPad 上

網址：**https://fringmin.github.io/kids_japanese/**

1. 用 Safari 開啟上面的網址
2. 分享按鈕 → 「加入主畫面」
3. 從主畫面圖示開啟即為全螢幕 App，之後離線也能玩
4. 「跟著唸」第一次使用會詢問麥克風權限，請按允許

## 開發

純 HTML/CSS/JS，無 build step。本機測試：

```bash
python3 -m http.server 8642
# 開 http://127.0.0.1:8642/
```

- 學習內容都在 `js/data.js`（假名例字、單字主題），要加單字改這裡就好
- 改版後記得把 `sw.js` 的 `CACHE_VERSION` +1，小孩的 iPad 才會更新快取
- 需求與設計記錄見 `REQUIREMENTS.md`

## 版權說明

（療癒系畫風：圓身體、豆豆眼、腮紅），以主題詞彙呼應孩子喜歡的角色世界觀。
