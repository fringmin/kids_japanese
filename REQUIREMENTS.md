# 需求文件：こどもにほんご — 兒童日文學習 PWA

> 建立日期：2026-07-10
> 狀態：✅ 已確認

## 問題與目標

- **要解決的問題**：兩個小孩（小三、小一）喜歡角落小夥伴、幽幽家族（オバケヌ）等日本角色，
  想看懂角色書上的日文（角色名、對話、圖旁的狀聲詞小字），也偶爾赴日旅遊，
  需要建立日文基本熟悉度。
- **期望改善**：用遊戲化方式讓他們自主反覆接觸 50 音與高頻單字，
  有聲音、能自己操作、不需大人陪讀。

## 使用者

- **主要使用者**：小三與小一的小孩（中文母語，注音能力有、英文拼音能力有限）
- **使用情境**：家中 iPad，Safari 加入主畫面後當 App 使用
- **介面語言**：繁體中文為主，日文內容附讀音與中文意思

## 功能需求（v1 Must Have）

- [x] **50 音表**：平假名＋片假名可切換，點任一假名就發音；
      每個假名附羅馬拼音小字與代表單字（例：ね→ねこ🐈）
- [x] **假名小遊戲**：聽音選字（播放「か」→ 從 4 個假名中選）、平片假名配對（あ↔ア）
- [x] **主題單字卡**（點卡片發音，圖示用 emoji／原創插圖）：
  - 角落生物主題詞：しろくま、ねこ、とんかつ、とかげ、ざっそう、すみっこ、かわいい…
  - 幽靈／幽幽家族主題詞：おばけ、こわい、よる、ばける…
  - 狀聲詞・擬態語（繪本圖旁小字最常見）：ドキドキ、ワクワク、ふわふわ、ぴかぴか…
  - 旅遊日語：こんにちは、ありがとう、すみません、おいしい、トイレ、でんしゃ…
- [x] **單字聽力遊戲**：聽單字選正確的圖
- [x] **跟讀模式（Speak 式）**：App 發音 → 小孩對麥克風跟著唸 →
      日文語音辨識比對給回饋。鼓勵導向：辨識成功大力稱讚；失敗可重試、
      也可自評過關，不因辨識問題挫折。辨識不可用（無網路／未授權）時
      自動退成「錄音回放自己聽」模式。
- [x] **得分機制**：每遊戲計分（答對得分＋連續答對 combo 加成），
      累積星星；記錄各遊戲最高分。
- [x] **雙人進度檔案**：開頭選人，各自累積星星、得分與已學會的假名/單字
      （localStorage 本機儲存）
- [x] **PWA**：可加入主畫面、離線可玩（介面與資料快取；發音用 iOS 內建語音，離線可用）

## Should Have（v2 之後）

- 簡短句子／對話模組（角色書常見句型）
- 濁音・半濁音・拗音（が、ぱ、しゃ…）專門關卡
- 書寫筆順練習

## Won't Have（本次不做）

- 不使用角落小夥伴／オバケヌ的官方圖片・美術素材（版權考量，
  GitHub Pages 屬公開網站），改用 emoji 與原創簡單圖案，
  僅以「主題詞彙」呼應他們喜歡的世界觀
- 不做帳號登入、雲端同步、排行榜

## 非功能需求

- **平台**：iPad Safari（PWA），手機與電腦瀏覽器也能開
- **語音輸出**：Web Speech API（speechSynthesis）呼叫 iOS 內建日文語音，
  做成可抽換模組，之後不滿意可換預錄音檔
- **語音輸入**：Web Speech API（SpeechRecognition, ja-JP），需網路與麥克風授權
- **操作**：按鈕大、適合兒童手指；一年級不識羅馬字也能純靠聽音操作
- **部署**：GitHub Pages（fringmin/kids_japanese），HTTPS
- **技術**：純 HTML/CSS/JS，無框架、無 build step，維護簡單

## 驗收標準

- [ ] iPad Safari 開啟 → 加入主畫面 → 從圖示啟動全螢幕運作
- [ ] 點 50 音表任一假名，1 秒內聽到日文發音
- [ ] 一年級小孩不需大人協助能完成一輪「聽音選字」並拿到星星
- [ ] 關掉重開後，兩人的星星與進度都還在
- [ ] 飛航模式下（已載入過）仍可開啟遊玩（跟讀辨識除外）

## 已知限制與風險

- iOS 的 Web Speech 需使用者先點擊一次才能發聲（以「開始」按鈕觸發）；
  不同 iOS 版本日文語音品質略有差異
- 語音辨識需網路（iOS 走 Siri 伺服器），對兒童發音辨識率不完美 →
  設計為鼓勵導向，不做嚴格判定
- localStorage 若清除 Safari 資料，進度會消失（此用途可接受）

## 技術備註

- 單頁應用（SPA）：`index.html` + `css/` + `js/`，畫面用 section 切換
- `js/data.js`：假名表與單字資料（唯一資料來源，日後擴充只改這裡）
- `js/speech.js`：發音模組（抽換點）；`js/recognition.js`：辨識模組（含 fallback）
- Service worker cache-first；改版時 bump `CACHE_VERSION`
